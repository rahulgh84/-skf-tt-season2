Season 2 Rules

Group Stage

Players are assigned to groups. Group matches are round-robin within the same group only.

Qualification

Manager can select Top 2, Top 3, or Top 4 from each group for knockouts. For two groups with Top 2, semifinals are A1 vs B2 and B1 vs A2.

Match Format

Best of 3 games. Each game is to 11 points. Win by 2.

Basement Rule

If the ball hits the ceiling or any basement obstruction, there is no reserve. Point goes to the opposite player.

Mandatory Service Rules

The server must toss the ball upward before contact. The ball must be visible and not hidden. Serve starts behind the end line and must touch server side first, then opponent side.
