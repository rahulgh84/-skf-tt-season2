Season 2 Rules

Match Format

Best of 3 games. Each game is to 11 points. Win by 2.

Basement Rule

If the ball hits the ceiling or any basement obstruction, there is no reserve. Point goes to the opposite player/team.

Mandatory Service Rules

The server must toss the ball upward before contact. The ball must be visible and not hidden. Serve starts behind the end line and must touch server side first, then opponent side.
